A simple app to clone (Read & Write EEPROM data), or set a new Odometer value. V2 now supports ZE0 and AZE0 clusters. V2.10 fixed some rounding errors in my math, it is now accurate to the Km.

Requires a J2534 passthrough device (Connect to your obd2 port).

I recommend an aliexpress Godiag GD101 or VNCI Nano passthrough.
